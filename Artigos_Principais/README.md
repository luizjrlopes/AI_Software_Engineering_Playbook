# Série Engenharia de Software com Agentes de IA

Este pacote organiza os 70 artigos planejados para publicação no GitHub, LinkedIn e demais canais profissionais.

## Estrutura

Os artigos estão divididos em nove blocos editoriais e oito arcos narrativos.

Cada arquivo contém número, título, bloco, arco narrativo, objetivo, tese central, desenvolvimento obrigatório, exemplo sugerido, artefato aplicável, critérios de senioridade, cuidados didáticos, pontos a evitar, fechamento, conceito de capa e status editorial.

## Status padrão

`PLANEJADO → PESQUISANDO → ROTEIRIZANDO → REDIGINDO → REVISANDO → PUBLICADO`

## Ordem de publicação

A ordem numérica é também a ordem narrativa recomendada.
